# SURF BUDDY
---
### A Full Stack web application that allows you to check current surfing conditions based on your location.
Check it out [here](https://stark-castle-27893.herokuapp.com/)!



## Technologies Used: 
1. JavsScript 
2. Node.js
3. MongoDB
4. Mongoose 
5. Express 
6. EJS
7. Method-Override
8. Semantic UI
9. Google Maps API
10. Spitcast API
